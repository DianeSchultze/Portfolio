Wildrose Foundry

A full-stack ecommerce platform for a craft metal foundry

Live Site

The Problem

Wildrose Foundry needed an ecommerce platform that could present products clearly, support a more organized shopping experience, and manage product data in a structured way instead of relying on static content alone.

The Solution

I designed and developed a full-stack ecommerce website that combined a custom front end with a database-driven back end, allowing products and content to be stored, managed, and displayed dynamically.

Tech Stack

  • HTML5 and CSS3 for structure and styling
  • JavaScript for interactivity
  • PHP for server-side development
  • MySQL for database design and management

Key Features

  • Dynamic product listings
  • Database-driven content
  • Structured ecommerce layout
  • Full-stack implementation

Process

This project was the final project for third semester at SAIT in the Web Design and Development program. It was developed as a full-stack ecommerce platform that brought together planning, database design, and front-end development.

The process began with building an ERD to map relationships between the different parts of the platform and understand how product data would be stored and connected. From there, the project moved into creating the database, inserting data, and connecting that data to the website through PHP and MySQL.

On the front end, the goal was to create an ecommerce experience that felt clear, functional, and visually aligned with the Wildrose Foundry brand. On the back end, the focus was on building a structure that could support real product information and dynamic content across the site.

Outcome

The final result is a database-backed ecommerce platform that demonstrates both technical development and visual design. It reflects the full process of creating a dynamic web experience, from ERD planning and database setup to implementation and final presentation.
