DORC ✿

A MERN stack daily quest tracker created for Web Security Fundamentals at SAIT

GitHub Repo

The Problem

DORC was created as a group project for Web Security Fundamentals at SAIT. The challenge was to build a full-stack application that was both engaging to use and grounded in secure development practices.

The Solution

Our team developed a daily quest tracker using the MERN stack, combining MongoDB, Express, React, Node.js, JSX, and Tailwind CSS. The application was designed to feel interactive and fun while still prioritizing security across the entire experience.

Tech Stack

  • MongoDB for data storage
  • Express.js and Node.js for the back end
  • React and JSX for the front end
  • Tailwind CSS for styling

Key Features

  • Daily quest tracking system
  • Responsive full-stack application
  • Sanitized and validated input fields
  • Encrypted handling of sensitive data

Process

This project was developed collaboratively with Chris Nordrum and Owen Ouyang. Working as a team meant planning both the front-end and back-end structure carefully, while making sure the application stayed consistent across features and components.

A major focus throughout development was security. Sensitive data was encrypted, user input was sanitized, and validation was applied to help protect the application from common vulnerabilities. This made the project a strong exercise in combining full-stack development with secure coding practices.

Outcome

DORC reflects my ability to contribute to a collaborative MERN stack project while applying practical web security fundamentals. The final result is an interactive application that balances functionality, structure, and protection.
